# AGENTS.md

모든 AI 에이전트는 아래를 필수로 따른다.

## Hard Rules

- 구현/분석 전 문서 확인은 필수다.
- 구현 시작 시 기본적으로 여는 문서는 최대 2개다.
  1. `docs/AGENT-CONTEXT.md`
  2. 작업 유형에 맞는 `docs/agent/*.md` 브리프 1개
- 막힐 때만 full guide 1개를 추가로 열 수 있다. 총 상한은 3개다.
- 구현 작업에서 `docs/README.md`는 기본 진입점으로 읽지 않는다.
- 새 주제는 새 스레드에서 시작한다.
- 중간 보고는 정말 필요한 경우만 1줄로 한다.
- 최종 보고는 `무엇을 바꿨는지 + 검증 결과`만 짧게 말한다.
- 문서와 코드가 어긋나면 같은 턴에서 같이 수정한다.

## Fast Path

- 항상 먼저: `docs/AGENT-CONTEXT.md`
- 그 다음: 작업 유형에 맞는 `docs/agent/*.md` 브리프 1개
- full guide는 브리프로 해결이 안 될 때만 추가
